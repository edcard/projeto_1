Estrutura Básica do HTML

HTML é uma linguagem de marcação de texto utilizada na estruturação de páginas web.

Sua sintaxe é bastante simples e é baseada em tags que compõe a sua estrutura. Elas representam os diversos elementos de uma página, como imagens e links.As tags são necessárias para que o documento seja corretamente interpretado pelos browsers.

Ao acessar uma página web através de um navegador, ele é capaz de interpretar o código HTML e renderizá-lo de forma compreensível para o usuário final, exibindo textos, botões etc. com as configurações definidas por meio das diversas tags que essa linguagem dispõe.

Atualmente a HTML encontra-se na versão 5 e é padronizada pelo W3C (World Wide Web Consortium), uma organização internacional responsável por estabelecer padrões para a internet.

Tags

(voltar)

Tags são elementos fundamentais que definem a estrutura e o conteúdo de uma página web. Elas funcionam como "etiquetas" que indicam ao navegador como exibir e interpretar o conteúdo da página. Basicamente, as tags são palavras-chave envoltas em sinais de menor e maior (< e>).

Atributos das Tags

(voltar)

Atributos em HTML são usados para fornecer informações adicionais sobre elementos HTML, configurando seu comportamento ou especificando detalhes sobre o seu conteúdo. Eles são inseridos dentro da tag de abertura do elemento e consistem em um nome e um valor.

Tipos de Atributos:

Globais

Atributos que podem ser usados em qualquer tag HTML. Exemplos incluem class, id, style, title, lang.

Específicos

Atributos que são aplicáveis apenas a tags específicas. Exemplos incluem src (para imagens e outros recursos), href (para links), alt (para texto alternativo em imagens), disabled (para desabilitar elementos).

Listas Ordenadas e Não Ordenadas

(voltar)

Em HTML, listas ordenadas e não ordenadas são usadas para organizar itens em uma página da web. Listas ordenada mostram itens com números ou outros marcadores sequenciais, enquanto listas não ordenadas usam marcadores (como pontos) para indicar os itens, sem uma ordem específica.

Listas Ordenadas

São usadas quando a ordem dos itens é importante, como em instruções passo a passo ou em um ranking.

Cada item da lista é marcado com um número, letra ou algarismo romano, por padrão, mas pode ser personalizado.

A tag ol define a lista, e a tag li (list item) define cada item da lista.

Listas Não Ordenadas

São usadas quando a ordem dos itens não é relevante, como em uma lista de ingredientes ou uma lista de tópicos.

Cada item da lista é marcado com um marcador (ponto, círculo, etc.), por padrão, mas pode ser alterado.

A tag ul define a lista, e a tag li (list item) define cada item da lista.

Links

(voltar)

Os links permitem que os usuários “viajem” de uma página para outra. Você sabe que esses links estão ativos quando o cursor do mouse mudar de uma seta para o ícone de uma mão ao passar o mouse sobre o link. Você pode anexar links HTML a vários elementos da página, incluindo texto e imagens

Para cirar link deve-se usar a tag a(ancôra) com o atributo href com o endereço para onde o link deve apontar. O texto visível do link é colocado entre as tags de abertura e fechamento.
